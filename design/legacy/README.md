# Design Legacy Archive

This folder keeps design artifacts that should not sit next to the active
redesign references.

Keep active source-of-truth files in `design/`, including the current IA,
chat, composer, and design-system references. Move files here when they are:

- superseded variant explorations,
- one-off implementation notes tied to an old branch or preview URL,
- duplicate copies from previous directory layouts,
- unrelated prototypes kept only for historical reference.
