# User Intent Baseline

## 한 줄 정의

ARIS는 사용자가 **채팅 UI에 가까운 인터페이스**에서 에이전틱 코딩을 수행하도록 설계된 웹 클라이언트다.

## 반드시 지켜야 할 의도

1. 관제 콘솔이 아니라, 모던 생산성 앱에 익숙한 사용자에게 자연스러운 경험을 제공한다.
2. 흰색 베이스 UI 위에서 의미 기반의 다채로운 컴포넌트로 정보를 전달한다.
3. AI 응답은 종류별(텍스트, 명령 실행, 코드 읽기, 코드 작성)로 분리된 UI 컴포넌트로 보여준다.
4. 사용자 의도 입력(모드, intent, constraint, 액션)은 빠른 조작 가능한 컴포넌트로 제공한다.
5. UI로 대응 불가한 상황을 위해 SSH fallback 옵션을 제공한다.
6. 로그인 기반 접근통제와 암호화/감사로그를 기본 보안으로 적용한다.
7. 모바일/태블릿/데스크톱에서 동일한 핵심 행동을 수행할 수 있어야 한다.

## UX 성공 기준

- 모바일에서 핵심 액션을 3탭 이내에 수행할 수 있다.
- 사용자는 5초 내 현재 세션 상태를 이해할 수 있다.
- 응답 타입을 시각적으로 혼동하지 않는다.
